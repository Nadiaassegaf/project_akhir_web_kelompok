beberapa hal yang harus di lakukan sebelum membuka website :
- pastikan database dan tabel-tabel nya sudah di buat, jika belum gunakan query builder di file queryBuilder->QUERYBUILDER.sql dengan mengeksekusinya pada mysql admin page.
- pastikan semua input value pada querybuilder sudah di eksekusi agar bisa masuk.

cara penggunaan website
- untuk pertama kali akan hanya muncul tampilan login, masukan akun yang sudah terdaftar, atau yang sebelumnya di inputkan ke database.
- jika belum punya akun atau lupa, bisa registrasi dengan menklik link registrasi. (registrasi dengan cara ini hanya akan set status ke user atau non admin, jika ingin memiliki status admin gunakan database, dan set status ke admin secara manual).
- setelah masuk, tinggal jelajahi website dengan menklik beberapa tombol page pada navigasi.
- klik logout bila ingin keluar dan mengakhiri sesi.
